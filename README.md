## 🛠️ Tech Stack

- **Kubernetes**
- **Istio**
- **Kafka**
- **Jaeger**

## 🚀 Quick Start

```bash
# Docker Compose (Recommended)
docker-compose up -d

# Local Development
pip install -r requirements.txt
python services/gateway.py
```

## 📖 Configuration

Edit `config/config.yaml` to customize the application.
